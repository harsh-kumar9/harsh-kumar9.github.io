# harsh-kumar.com (2026 rebuild)

Based on the template by Abhraneel Sarma (https://github.com/abhsarma/abhsarma.github.io),
also used by Sunnie S. Y. Kim. Bootstrap 3 + custom.css; no build step — push to GitHub Pages
or any static host as-is.

## Before publishing
1. Replace `img/profile.jpg` with your photo (square, ~600px).
2. Add your CV PDF at `about/harsh_kumar_cv.pdf` (filename referenced from index.html).
3. Search both HTML files for `TODO` — these mark missing links (EMNLP/ICML/CHI'26 papers),
   the exact ICML paper title, dates to confirm, additional invited talks, and your
   reviewing/service record.
4. Font note: the template ships Kepler Std and Avenir Next (Adobe/commercial faces).
   Both upstream sites ship them too, but if you want to be safe, swap in free
   near-equivalents (e.g. Source Serif 4 for Kepler, Nunito Sans for Avenir) in custom.css.
5. Template credit: consider keeping a footer line crediting Abhraneel, as Sunnie does.
